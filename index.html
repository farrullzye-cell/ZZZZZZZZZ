FAQ

Pertanyaan Umum

Jawaban atas pertanyaan yang sering diajukan tentang layanan kami.

Berapa lama waktu pengerjaan website?

Waktu pengerjaan tergantung pada kompleksitas project. Landing Page biasanya 3-7 hari, Company Profile 5-10 hari, dan website kompleks 2-4 minggu. Kami akan memberikan estimasi waktu yang jelas setelah konsultasi.

Apakah saya mendapatkan source code website?

Ya, Anda mendapatkan full source code website beserta database (jika ada). Semua hak kepemilikan source code sepenuhnya milik Anda.

Apakah bisa melakukan revisi?

Tentu! Kami memberikan gratis revisi sebanyak 2 kali untuk memastikan hasil sesuai keinginan Anda. Revisi tambahan dapat dilakukan dengan biaya terjangkau.

Apakah bisa request fitur tambahan?

Bisa. Kami menerima request fitur tambahan di luar paket. Tim kami akan menganalisa dan memberikan penawaran harga sesuai kompleksitas fitur yang diminta.

Apakah ada layanan maintenance?

Ya, kami menyediakan paket maintenance bulanan yang mencakup update keamanan, backup database, monitoring performa, dan support teknis.

Apakah bisa pembayaran DP?

Tentu. Sistem pembayaran kami menggunakan DP 50% di awal dan 50% setelah selesai. Untuk project besar dapat diangsur sesuai kesepakatan.

Siap Memiliki Website Profesional?

Hubungi kami sekarang dan konsultasikan kebutuhan website Anda secara GRATIS. Tim kami siap membantu Anda dari konsultasi hingga launching.
